# 🎈마크다운이란?  
---
Git 블로그를 운영하다보면 마크다운에 대해서 자연스레 접하게 된다.  
그렇다면 마크다운이란?  
**일반 텍스트 문서에 서식 요소**를 추가하는 데 사용할 수 있는 가벼운 마크업 언어  
이렇게 볼수있다.  
본 포스팅에서는 크게 기본 구문/확장 구문/기타 구문으로 나누어서 설명한다. 
| 구문 분류   | 설명                                    |
|-------------|-----------------------------------------|
| 기본 구문   | 글자 속성에 대한 마크업 구문             |
| 확장 구문   | 마크다운 구문을 기반한 고급 기능         |
| 기타 구문   | 이모지같은 특수한 기능 구현 구문         |
| 참고 사항   | 참고 사항                               |
